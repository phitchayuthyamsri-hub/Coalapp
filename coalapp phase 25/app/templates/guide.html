Guide

The corridor

XPPL Mine (Kaleum, Sekong, Laos) → Lalay border → QL49 highway → Chân Mây port (Vietnam). An A Ngo warehouse hub is a planned leg-decoupling point.

QL49 operating windows (the bottleneck)

  • Border crossing (loaded): 15:00–17:00, +1h paperwork
  • QL49 inbound (loaded, Mine→Port): 19:00–24:00
  • Port unload: 07:00–17:00
  • QL49 return (empty, Port→Mine): 00:00–05:00

Pages

  • Dashboard — upload data; live fleet status with ETA.
  • Fleet — register trucks, phone, status, effective dates.
  • Routes — draw each leg's road line + set speed. ETA needs these.
  • Timeline — each truck's zone visits over time.
  • Plan vs Actual — planned vs actual port arrival, on-time %.
  • Subcontractor — claimed vs GPS-estimated Mine arrival.
  • Performance / Daily — trip counts, cycle time, per-day throughput.

Data sources

GPS xlsx (vehicle pings), Dispatch Plan xlsx, Weighbridge (Load_actual) xlsx, Subcontractor xlsx. Weighbridge is the authority for loading; GPS is the secondary check.
